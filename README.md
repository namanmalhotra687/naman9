# 📋 Naman TaskBoard

A FastAPI-based task manager with:

- 🔐 Login/logout
- 📝 Add/edit/delete tasks
- 📊 Dashboard (Chart.js)
- ⚙️ Task generator (/generate)
- 🔁 Recurring tasks (APScheduler)
- 🔎 Search/filter
- ⬇️ Export to CSV
- 🕒 Auto logout on inactivity

🚀 Built with FastAPI, Jinja2, Bootstrap 5
